# DOKUMENTASI

## Live Demo: https://pokemon-siroj.netlify.app/pokemon-list

## Repository: https://github.com/siroj05/Pokemon-Frontend

## TECH STACK
- Next.js -> Framework & routing
- Tailwind CSS -> Styling UI
- TanStack Query -> Data fetching
- Axios -> HTTP client
- Zustand -> State management

## STRUKTUR FOLDER
- app/ ---------> routing dan halaman menggunakan nextjs App router
- components/ --> folder komponen yang reusable
- hooks/ -------> untuk custom hook
- services/ ----> untuk layer api menggunakan axios
- store/ -------> state management global menggunakan zustand
- config/ ------> konfigurasi provider Tanstack query Provider
